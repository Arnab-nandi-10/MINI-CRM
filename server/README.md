# Mini CRM System - Server (Backend)

Node.js + Express REST API with MongoDB (Mongoose), JWT auth, role-based access, Nodemailer, and Twilio/Fast2SMS integration.

## Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- JWT Auth
- Nodemailer
- Twilio/Fast2SMS

## Modules
- Client Profile Management
- Lead Tracking
- Task & Meeting Scheduling
- Email/SMS Logging
- Dashboard Analytics
- Role-Based Access Control

## Setup
1. Install dependencies: `npm install`
2. Configure `.env` for DB, JWT, email, SMS
3. Start server: `npm run dev` or `npm start`

API docs and Postman collection included.
