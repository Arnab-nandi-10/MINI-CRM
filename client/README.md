# Mini CRM System - Client (Frontend)

This React.js app uses Tailwind CSS, Axios, React Router, and Chart.js/Recharts for a modern, responsive UI. It connects to the backend REST API for all CRM operations.

## Tech Stack
- React.js
- Tailwind CSS
- Axios
- React Router
- Chart.js or Recharts

## Modules
- Client Profile Management
- Lead Tracking
- Task & Meeting Scheduling
- Dashboard Analytics
- Auth (JWT, role-based)

## Setup
1. Install dependencies: `npm install`
2. Start dev server: `npm start`

Configure API endpoints in `.env` as needed.
